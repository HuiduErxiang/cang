# 藏经阁 - 知识仓真相源

> **定位**: 独立知识仓 + 蒸馏运行地 + 发布物产出地

## 目录结构

```
藏经阁/
├── raw/              # 原始来源
├── staging/          # 提取与清洗中的中间产物
├── l1/               # 写作心法层
├── l2/               # 医学策略层
├── l3/               # 疾病通识层
├── l4/               # 产品证据层
├── lineage/          # 血缘、状态、审核记录
├── publish/          # 对外发布快照
│   ├── current/      # 当前发布视图
│   └── releases/     # 历史发布快照
├── exporters/        # 面向消费者的兼容导出器
│   ├── v2/
│   └── xiakedao/
├── manifests/        # 清单、版本与发布说明
├── tools/            # 蒸馏与导出工具
└── docs/             # 知识仓内说明文档
```

## 核心原则

1. **知识真相源**: 所有正式知识以藏经阁为唯一长期归属
2. **蒸馏链在仓内运行**: 素材吸纳、片段提取、事实构建、晋升、发布在仓内完成
3. **消费方只读**: V2 和侠客岛只能消费已发布知识，不反写
4. **发布物与真相源分离**: 消费系统读取 publish/ 下的发布快照

## 知识层级定义

| 层级 | 名称 | 内容 |
|------|------|------|
| L1 | 写作心法层 | 通用写作规则、表达原则、风格 DNA |
| L2 | 医学策略层 | 编辑策略、医学 playbook、写作规范 |
| L3 | 疾病通识层 | 疾病知识、病程、机制、诊疗共识 |
| L4 | 产品证据层 | 产品证据、试验设计、疗效、安全性 |

## 与消费系统的关系

- **V2**: 读取 `publish/current/consumers/v2/`
- **侠客岛**: 读取 `publish/current/consumers/xiakedao/`
- **前端**: 不直连藏经阁，只通过侠客岛访问
