## 1. 验证方法

### 1.1 蒸馏工具链

| 组件 | 版本/路径 | 说明 |
|------|-----------|------|
| Kimi CLI | v1.24.0 | --print --final-message-only --output-format text |
| Python 管道 | _distill_clean.py | subprocess 捕获 stdout，绕过 PowerShell 编码损坏 |
| JSON 提取 | 三级策略 | TextPart regex → ToolResult regex → 花括号匹配 |
| 超时 | 600s | 大文件（>5MB）需要更长处理时间 |

### 1.2 样本选取

从 316 个 PDF 中按四种文档类型各选 3 份代表性样本（共 12 份）。选材覆盖中英文、大小文件、不同出版来源。

### 1.3 验证标准

1. json.load() 无异常
2. 公共字段（title/source/year/language/total_pages/doi）均存在且合理
3. 类型自适应字段（如 trial_name、issuing_body、data_source、conference_name）均已填充
4. figures/tables 锚点（如有）含页码和描述
